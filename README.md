
# WebNexus App Shell - מדריך הפקה לגרסת HTML

הפרויקט מוגדר להפקה של קבצים סטטיים (HTML, CSS, JS) שניתן להריץ ללא שרת, מה שמתאים מאוד ליצירת אפליקציות חסומות לילדים.

## איך משיגים את קבצי ה-HTML?

1. **הורדה:** לחץ על "Download as ZIP" בממשק והוצא את הקבצים למחשב שלך.
2. **התקנה:** פתח טרמינל בתיקייה והרצ: `npm install`
3. **בנייה:** הרץ את הפקודה הבאה:
   ```bash
   npm run build
   ```
4. **התוצאה:** לאחר סיום הפעולה, תיווצר לך תיקייה חדשה בשם `out`.
   - **תיקיית `out` היא האפליקציה שלך!**
   - היא מכילה קובץ `index.html` וכל מה שצריך כדי שהאפליקציה תעבוד.

## איך להפוך את זה לאפליקציה ללא דפדפן?

### אופציה א': Capacitor (מומלץ לאנדרואיד)
זה יהפוך את תיקיית ה-`out` ל-APK אמיתי:
```bash
npx cap add android
npx cap sync
npx cap open android
```

### אופציה ב': WebView פשוט
אם יש לך כבר מעטפת שקוראת קבצי HTML, פשוט כוון אותה לקובץ ה-`index.html` שנמצא בתוך תיקיית ה-`out`.

## הערה על בטיחות ילדים:
בממשק הניהול (לחיצה של 5 שניות בפינה השמאלית למטה), וודא ש"מצב קיוסק" פעיל. בגרסה המותקנת, זה ימנע מהילדים לצאת מהאפליקציה או לגלוש לאתרים לא מאושרים.
